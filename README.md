# ideabox

Application built with Node.js, Mongodb, Backbone.js

Features :

Participants can
    
    - Post new ideas
    - Upvote / Downvote ideas
    - Comment on ideas
    - Every idea will have a computed score based on the votes received
    
Screens
    
    - Home page
        
        - Ideas list
        - Filter by product, feature,
